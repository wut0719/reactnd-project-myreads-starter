# MyReads Project

项目6: MyReads Project。基于起始代码构建。应用功能：展示书架列表，更改图书状态，搜索图书

## 安装和启动项目

1. `git clone` 到本地项目目录
2. 在项目目录中，执行 `npm install` 安装依赖
3. 执行 `npm start` 启动项目前端服务器，在浏览器中使用

## 组件结构

- `ShelfSelect`，修改书籍状态的受控select组件
- `Book`，书籍组件
- `Bookshelf`，书架组件，包含多个书籍组件
- `Search`，搜索组件
